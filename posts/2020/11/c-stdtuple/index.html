C++ - std::tuple

November 27, 2020 One-minute read
C++

std::tuple was added in c++11 fixed sized heterogeneous values It is a generalization of std::pair. full details at cpp reference

get<index>(tuple) is used to get the values at index.

what is interesting about tuple though is using make_tuple and auto to create tuples without specifying types. auto type deduction will figure it out.

#include <iostream>
#include <utility>
#include <string>
#include <tuple> // for tuple
using namespace std;

int main()
{
    std::tuple<int, int, string> p;

    p = make_tuple(1, 2, "fff");

    cout << get<0>(p) << endl;

    auto p1 = make_tuple("1dd", 1.1);
    cout << get<0>(p1) << endl;
}
