C++ - Type alias

March 20, 2020 One-minute read
c++

for template classes, alias can be used to declare specialization of template class. I guess it can used to define default configuration of the template class.

#include <iostream>

template <unsigned T>
class cls
{
public:
  cls()
  {
    std::cout << T << std::endl;
  }
};

// The alias with T=32
using alias = cls<32>;

int main()
{
  alias c;
}
