Waveforms and WaveJSON - wavedrom and wavedrompy

May 21, 2022 One-minute read
DV

Wavedrom is a tool to render SVG of waveforms using WaveJSON. WaveJSON is JSON based format docs to documents signal values with labels and annotations. wavedrom is written in javascript but there is wavedrompy which is the python implementation.
