The one books everyone should read

February 21, 2023 One-minute read
Books

You may think there is a typo in the title. It is not! This post is about The one book I like the most in each topic. I mean if you have to read just one book in computer Architecture, What would it be?

Disclaimer, This is purely personal preference so don’t fight me if you don’t like these books.

  • Computer Organization: Digital Design and Computer Architecture, David Harris.
  • Computer Architecture: Computer Architecture: A Quantitative Approach
  • Cryptography: Understanding Cryptography
  • Computer Graphics: Computer Graphics Principles Practice
  • USB: Mindshare
  • PCIE: Again Mindshare
  • Graph Theory: Introduction to Graph Theory
  • Linux Kernel: Understanding The Linux Kernel
  • C : The C programming Language.
  • C++: Modern C++ Design
  • Python: Programming Python
  • Computation Theory: Introduction to the Theory of Computation
  • UVM: Advanced UVM
  • Systemverilog: systemverilog for verification
